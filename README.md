# Cryptography With Python

## symmetric Encryption

```
python symantric_encryption.py -e thushan
```

<p>output -> lJrfeRjNuw== </p>

## symmetric Decryption

```
python symantric_encryption.py -d lJrfeRjNuw==
```
<p>output -> thushan </p>

python asymmetric_encryption.py
